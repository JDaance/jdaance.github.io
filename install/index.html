Skyturns beta install

Remember this is a beta TEST. So you are a tester of the game and are supposed to report your experiences positive and negative :) If something strange happens while playing (running) you can pause the game and press "Dump replay". This will upload your run to our server and give us a chance to see exactly what happened and fix any problems.

Have fun! // J&R

Android

Just download and install from this link: skyturns.apk

iOS

Follow these steps:
  • First you have to supply us your device UDID for us to add to the app to make Apple happy. One way to get UDID is by visiting http://get.udid.io/ from your device, or following this guide through iTunes: http://whatsmyudid.com/.
  • Send your UDID to Joakim or Rebecka and we will add it to the app.
  • Wait a few hours until you download the app from this site. (After we get the UDID we have to update the code and upload a new version of the app to the site)


After we have manually added your UDID you can install the game from your device by following this link: Skyturns iOS install
(Latest update of the code: 2014-04-23 kl 18:15)

Troubles?

  • If iOS refuses to install make sure we have added your UDID to the app
  • On Android you might need to enable some setting to allow apps from outside Google Play
  • If Android install just fails, it's usually out of space. Worst case you might have to uninstall an older version of Skyturns first.
